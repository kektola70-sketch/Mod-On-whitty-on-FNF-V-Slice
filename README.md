# VS Whitty - FNF V-Slice Port

## Описание
Полный порт оригинального мода VS Whitty для Friday Night Funkin' V-Slice версии 0.8.1

## Особенности
- ✅ Все 3 оригинальные песни (Lo-Fight, Overhead, Ballistic)
- ✅ Уникальный персонаж Whitty
- ✅ Кастомная сцена (Alley)
- ✅ Динамические эффекты камеры
- ✅ Полная совместимость с V-Slice 0.8.1

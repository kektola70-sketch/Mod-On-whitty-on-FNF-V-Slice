# Установка VS Whitty для FNF V-Slice 0.8.1

## Способ 1: Автоматическая установка (рекомендуется)

### Для Windows:
1. Скачайте FNF V-Slice 0.8.1
2. Скачайте архив `whitty-mod.zip`
3. Распакуйте содержимое в папку `mods/`
4. Запустите `Funkin.exe`

### Для Android (через Acode):
1. Установите Acode из Play Store
2. Клонируйте репозиторий мода
3. Откройте папку FNF V-Slice
4. Скопируйте папку `whitty-mod` в `mods/`
5. Соберите APK через встроенные инструменты
